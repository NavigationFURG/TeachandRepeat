Parameters
==========

+-------------------+---------------+-----------------------------------------------------------------------------------------+
| Parameter         | Default Value | Description                                                                             |
+===================+===============+=========================================================================================+
| reference_frame   | map           | The reference frame for the path points.                                                |
|                   |               | If set to "map", points are relative to the map frame.                                  |
|                   |               | If set to "odom", points are relative to the robot's odometry frame.                    |
+-------------------+---------------+-----------------------------------------------------------------------------------------+
